## 어떻게 실행하나요

사전 조건: `wts-backend`가 `http://localhost:9789` 에서 실행 중이어야 합니다.

프론트엔드 개발 서버 실행:

```cmd
cd wts-frontend
npm install
npm run dev
```

- 브라우저에서 `http://localhost:5173/health` 로 접속하면 백엔드의 `/api/health` 응답을 확인할 수 있습니다.
- 개발 서버는 `/api`로 시작하는 요청을 자동으로 `http://localhost:9789` 로 프록시합니다(`vite.config.ts`).

프로덕션 빌드/미리보기:

```cmd
cd wts-frontend
npm run build
npm run preview
```

## 연동 포인트

- 프록시: `wts-frontend/vite.config.ts` – `server.proxy['/api']` → `http://localhost:9789`.
- API 클라이언트: `wts-frontend/src/api/client.ts` – `baseURL: '/api'` (개발 시 Vite 프록시 이용, 운영 시 리버스 프록시/환경변수로 조정).
- 헬스 페이지: `wts-frontend/src/pages/Health.tsx` – `GET /api/health` 호출 후 응답 문자열 출력.
